## How to install?

1. Run the following command:

```
npm install
```

2. Open `./alts/alts.txt` and upload the alts there like this

```
email:pass
email:pass
email:pass
email:pass
email:pass
```

3. Upload the files to your host

4. change the port on "config.json" to your port
